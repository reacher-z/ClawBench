## Guardrails applied

- **Account-tied channels I do NOT submit from:** HN, Reddit, X, LinkedIn, newsletter tips — those need to come from the user's own accounts for reputation reasons. All drafts are ready to paste.
- **GitHub PRs I do submit:** curated awesome-list entries only, not mass-spam. See [executed.md](executed.md).
- **Disclosure:** every PR discloses maintainer affiliation (reacher-z) per steel-dev and similar lists' CONTRIBUTING rules.
- **No co-author trailer** on any commit, per repo convention.
