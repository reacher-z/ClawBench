# 中文社区推广渠道

## 优先级排序

1. 机器之心 / 量子位 邮件投稿
2. @爱可可-爱生活 微博 (学术论文转发首选)
3. 知乎专栏 (一作自述格式)
4. PaperWeekly 投稿
5. V2EX /go/share
6. ModelScope dataset 上架
7. 掘金教程博客
8. HyperAI 超神经
9. 新智元
10. CSDN
11. 小红书 (如果想触达消费级用户)

## 1. 知乎 (zhihu.com)

**入口:** 专栏文章,话题 `#人工智能` / `#大语言模型` / `#AI Agent`. 回答类似 "如何评价 browser-use / OpenAI Operator?" 的现有问题效果更好.

**规则:** 新账号盐值低易被限流. 以 **第一作者自述** 格式发文("作者自述:我们为什么做 ClawBench")是知乎最能被算法接受的自我推广形式. 建议完成 学者认证 (ORCID/Google Scholar). 同一链接 24 小时内回答 >3 个问题会触发广告检测.

**文案 (CN):**
> 我们刚开源了 ClawBench——153 个真实世界网页任务构成的 AI 浏览器 agent benchmark,涵盖购物、订票、表单、信息检索等日常场景。相比 WebArena 的沙盒环境,我们直接测真实站点,并开放了 Chrome 插件让社区一键跑评测。欢迎大家 star、提 PR,也期待讨论 agent 评测该不该走出沙盒。

## 2. V2EX (v2ex.com)

**节点:** `/go/share` (项目分享,最宽容) 或 `/go/create` (作品首发). 避免 `/go/dev` — 会被举报为广告.

**规则:** 发帖需积分 (建议 >100). 非广告口吻 — 讲故事 / 动机 / 技术难点, 项目链接放文末. 时间: 工作日 10:00 或 21:00 北京时间.

**文案 (CN):**
> 折腾了半年,开源一个 AI 浏览器 agent 的 benchmark:153 个日常任务(下单/订机票/填表),直接跑真实网站而不是模拟器。我们测了 7 个前沿模型,最高通过率 33.3%,绝大多数不到 40%。求 V 友来怼 baseline。GitHub: reacher-z/ClawBench

## 3. 机器之心 (jiqizhixin.com)

**风格:** 严肃学术 + 行业含义, 3000-5000 字, 一作署名. 喜欢 paper + 开源代码 + 榜单三件套.

## 4. 量子位 (qbitai.com)

**投稿邮箱:** `ai@qbitai.com`

**风格:** "XX 大学团队开源 XXX" 标题党, 需要 1-2 个亮眼数字 (如"最强模型也只有 33.3% 通过率"). 响应更快.

## 5. 新智元 (aiera.com.cn)

**投稿邮箱:** `aiera@aiera.com.cn`

**风格:** 硬核 + 国际视野, 喜欢和 OpenAI Operator / Anthropic Computer Use 对比.

## 机器之心/量子位/新智元 通用 pitch (CN)

> UBC、Vector Institute、清华、港科大、浙大、上交、CMU、Waterloo 等机构联合 Etude AI / UniPat AI / Netmind.ai 推出 ClawBench:首个聚焦真实网页的 AI 浏览器 agent 评测基准。153 个任务覆盖电商、出行、政务、办公场景,配套 Chrome 插件可一键复现。我们评测了 7 个前沿模型,最高通过率仅 33.3%,绝大多数不到 40%,暴露 DOM 噪声与多步推理的硬伤。Paper: huggingface.co/papers/2604.08523,Repo: github.com/reacher-z/ClawBench

## 6. 掘金 (juejin.cn)

**Tag:** 人工智能 + 开源 + AI 编程 + Chrome 插件

**风格:** 技术博客式, 2000-3000 字, 必须有代码片段和截图. 算法偏工程实战 — 要讲怎么用.

**文案 (CN):**
> 手把手带你用 ClawBench 评测自己的浏览器 agent:装 Chrome 插件、跑 153 个真实任务、在 leaderboard 上打榜。附完整 Python SDK 示例。

## 7. CSDN (csdn.net)

**优先级:** 低. SEO 仍强 (百度搜索友好), 但研究者社区质量一般.

**Category:** 人工智能 / 大模型 / 浏览器扩展开发. 标题可加"保姆级"或"踩坑总结".

## 9. HF 中文 / ModelScope (modelscope.cn)

**文案 (CN):**
> ClawBench 评测集已上线魔搭,支持一键 `modelscope download`. 中文真实站点任务占比 40%+, 欢迎复现.
