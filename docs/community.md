## Discord (English)

**Invite:** https://discord.gg/clawbench

### Channels

| Channel | What goes there |
|---|---|
| `#announcements` | Maintainer-only — new releases, leaderboard updates, paper revisions |
| `#general` | Open discussion about browser agents, benchmarks, the state of the art |
| `#contributors` | Coordination for people working on test cases and framework changes |
| `#model-submissions` | Help with getting a new model onto the public leaderboard |
| `#help` | Running into an install or container-engine problem? Ask here |
| `#research` | Paper discussion, related work, evaluation methodology |
| `#showcase` | Demo videos, GIFs, interesting failure modes |

### House rules

1. Be decent. No harassment, personal attacks, or slurs.
2. No promotion of unrelated projects without asking first. Sharing related research or tools is welcome in `#general` and `#research`.
3. No scraping the server or mass-DMing members. Kick on first offense, ban on second.
4. Moderation follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). Maintainers reserve the right to remove posts and ban accounts.

### How to get contributor role

Open a merged PR to ClawBench (any kind — test case, fix, doc). Post the PR link in `#contributors` and a mod will grant the role. Contributors get write access to `#contributors` and a vote on open design questions.

### 群聊主题

- ClawBench 新闻与版本更新
- 浏览器 agent 研究分享(论文 / 博客 / demo 视频)
- 测试用例贡献讨论(如何写 eval_schema, 踩过的坑, 评审协作)

### 群规

1. **实名或昵称 + 机构**:入群请改群名片为 "真实姓名 or 昵称 @ 学校/公司"。
2. **不发广告**:禁止群发商品、付费课程、不相关的拉群链接。
3. **不转发无关八卦**:保持话题与 AI agent / benchmarking / 浏览器自动化 相关。
4. **保护隐私**:不在群里公开他人联系方式、薪资、学校录取结果。
5. **违反规则 3 次以上踢出**,严重违规(骚扰、网暴)直接踢出并拉黑。

## GitHub Discussions

Not on chat? Open an [issue](https://github.com/reacher-z/ClawBench/issues/new/choose) — we respond fastest there.

We follow the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) across all channels (Discord, 微信群, GitHub). Maintainers enforce it.
