# Community assets

Drop the following files here when the groups are created:

- `wechat-qr.png` — 500×500 or larger PNG of the 微信群 entry QR (either a live group QR code or a personal-account QR that redirects to the group). WeChat group QRs expire after 7 days — rotate the file weekly or switch to the personal-account redirect flow.
- `discord-invite.txt` — optional plain-text Discord invite code (e.g., `clawbench`), not the full URL.

Both are referenced by [docs/community.md](../../docs/community.md) and the README Community section.
